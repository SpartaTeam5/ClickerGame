
## 프로젝트 소개
# ClickerGame
내일 배움 캠프에서 진행한 팀 프로젝트 클리커 게임 입니다

## 게임 설명
꼬북단들의 모험

## 개발기간 : 2025.03.27 ~ 2025.04.02

## 사용엔진 : UnityEngine 2022.3.17f1 (LTS)

## 언어 : C#

## 팀 소개
팀장 김태겸 

팀원 김규태, 나진영, 이재휴

## 플레이방법
클릭하나로 돈을 벌고 전투를 하며 무기와 능력치를 성장시켜 더욱 강한 포켓몬을 쓰러트리는 모험!
